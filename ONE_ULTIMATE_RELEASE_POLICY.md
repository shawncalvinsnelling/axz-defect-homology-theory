# One Ultimate Release Policy

This repository is intended to be published as one complete public artifact, not as a confusing sequence of partial public versions.

## Public naming

Use:

```text
AXZ Defect Homology Theory — Ultimate Release
```

Avoid leading with staged labels like V1, V2, V3, or V4 in public descriptions.

## Recommended GitHub tag

```text
ultimate
```

## What stays public

- theorem statement
- finite verifier
- examples
- receipts
- schemas
- reproducibility guide
- truth boundary

## What should not be overclaimed

- no global RH proof
- no global P vs NP proof
- no global Erdős–Straus proof
- no global Navier–Stokes proof
- no global Hodge or Beal proof

## Public statement

This is a finite proof-audit framework for computing obstruction classes and replaying receipt evidence in bounded systems.
