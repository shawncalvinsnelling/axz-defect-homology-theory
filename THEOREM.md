# The Snelling Full Chain Defect Homology Theorem

Let

\[
K=(C,R,G,H,\partial_1,\partial_2,\partial_3)
\]

be a finite AXZ chain complex over \(\mathbb F_2\):

\[
C \xrightarrow{\partial_1} R \xrightarrow{\partial_2} G \xrightarrow{\partial_3} H
\]

satisfying:

\[
\partial_2\partial_1=0,\qquad \partial_3\partial_2=0.
\]

Define:

\[
H_R^{AXZ}(K)=\ker(\partial_2)/\operatorname{im}(\partial_1)
\]

and:

\[
H_G^{AXZ}(K)=\ker(\partial_3)/\operatorname{im}(\partial_2).
\]

Then a finite AXZ audit is fully closed exactly when:

\[
\dim H_R^{AXZ}(K)=0
\quad\text{and}\quad
\dim H_G^{AXZ}(K)=0.
\]

If \(H_R^{AXZ}\ne0\), unexplained residual defect classes remain. If \(H_G^{AXZ}\ne0\), unrecepted gate obligations remain.

Boundary: this is a finite proof-audit theorem. It does not prove any global open conjecture by itself.
