# Proof Sketch

1. \(\partial_2\partial_1=0\) gives \(\operatorname{im}(\partial_1)\subseteq\ker(\partial_2)\), so \(H_R^{AXZ}\) is well-defined.
2. \(\partial_3\partial_2=0\) gives \(\operatorname{im}(\partial_2)\subseteq\ker(\partial_3)\), so \(H_G^{AXZ}\) is well-defined.
3. \(H_R^{AXZ}=0\) means no independent residual defect class remains.
4. \(H_G^{AXZ}=0\) means no independent unrecepted gate obligation remains.
5. If every active gate satisfies \(\varepsilon_i<\Gamma_i\), replay error cannot flip the truth label.
